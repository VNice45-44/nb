Current architecture notes
- IndexPage.vue owns the visible notebook/library layout.
- useNotebookWorkspace.js owns notebook state, engineering library data, selections, computed lookups, new-entry creation, and terminal commands.
- app.scss owns the notebook, library panel, machine rail, and terminal dock styling.
- MainLayout.vue provides the Quasar shell around the page.

Deployment notes
- Quasar build command: npm run build
- Static output: dist/spa
- Netlify publish directory: dist/spa
